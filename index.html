🐍 Snake Game

Classic Snake game implemented in Python with both terminal and web versions!

🌐 Play in Browser

Features

  • 🖥️ Terminal version with keyboard controls
  • 🌐 Web version powered by Pyodide
  • 🎮 WASD and arrow key controls
  • ⚡ Adjustable game speed (+/- keys)
  • 🎯 Score tracking
  • 🐍 Growing snake mechanics
  • 💥 Collision detection

Controls

  • Move: Arrow keys or WASD
  • Speed up: + or = key
  • Speed down: - or _ key
  • Quit: Q key
